# SkullCandy Component Library

Welcome to the SkullCandy NFT Marketplace component library! This Storybook contains all the components used in the SkullCandy project.

## üé® Design System

All components follow the SkullCandy design system with:
- **Glass morphism** UI patterns
- **Orbitron** font for headings
- **Outfit** font for body text
- **Gradient accents** and borders
- **Dark theme** optimized

## üìê Design Tokens

Key design tokens used across components:

### Typography
- `font-family-title` - Orbitron (headings)
- `font-family-body` - Outfit (body text)

### Colors
- `--glass-light` - Light glass morphism background
- `--glass-dark` - Dark glass morphism background
- `--button-gradient` - CTA button gradient

### Effects
- `--glass-blur` - Backdrop blur amount (100px)
- `--gradient-border-main` - Glass border gradient

### Spacing
- Standard Tailwind spacing scale
- Custom gaps for grid layouts

## üéØ Best Practices

1. **Use semantic tokens** - Always use design tokens instead of hardcoded values
2. **Follow Figma specs** - Components match Figma designs pixel-perfect
3. **Accessibility first** - All components are keyboard accessible and screen reader friendly
4. **Responsive by default** - Components adapt to different screen sizes
5. **TypeScript interfaces** - All props are fully typed
